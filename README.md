# Signals and Linear Systems (CSE 220 Sessional)

This repository contains my coursework, assignments, practice problems, and project implementations for **CSE 220: Signals and Linear Systems Sessional**.

The course focuses on understanding continuous and discrete-time signals, analyzing linear systems, and implementing signal processing techniques using Python.

---

## 📚 Course Information

- **Course:** CSE 220 – Signals and Linear Systems Sessional
- **Department:** Computer Science and Engineering (CSE)
- **Language:** Python
- **Primary Textbook:** *Signals and Systems* by Alan V. Oppenheim and A. S. Willsky

---

## 🎯 Course Objectives

- Understand continuous-time and discrete-time signals.
- Learn properties of linear systems.
- Study the sampling process.
- Apply Fourier Series, Continuous Fourier Transform (CFT), Discrete Fourier Transform (DFT), Fast Fourier Transform (FFT), Laplace Transform, and Z-Transform.
- Analyze signals in both time and frequency domains.
- Implement signal processing algorithms using Python.
